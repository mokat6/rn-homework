# Install

git clone
cd rn-homework
npm install

# Run

npm start

Follow terminal instructions.

Open in `Expo go` you need to have the `Expo Go` app on your physical device. Then scan the QR code

Open in simulator - hit "a" for Android, "i" for iPhone
